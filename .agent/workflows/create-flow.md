# Criar Novo Fluxo de Bot
1. Crie um arquivo JSON em `src/blueprints/flows/[nome].json`.
2. Defina os estados (nodes) do bot.
3. Utilize nomes de Molecules ou Organisms existentes no campo "perform" ou "action".
4. Verifique se as variáveis {{template}} estão de acordo com o `parser.ts`.
